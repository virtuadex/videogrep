VoxGrep

VoxGrep is a command line tool maintained by virtuadex (originally by Sam Lavigne) that searches through dialog in video files and makes "supercuts" based on what it finds. It will recognize .srt, .vtt, or .json subtitle tracks, and transcriptions that can be generated with Whisper or pocketsphinx. It's like grep, but for video.

For installation, usage instructions, and code, see: https://github.com/virtuadex/voxgrep.

Or read the tutorial.

Roadmap

  • Semantic Search: Search by concept using vector embeddings.
  • Speaker Diarization: Filter results by specific speakers.
  • Embedded Subtitles: Burn-in subtitles directly onto the supercut.
  • Web GUI: FastAPI backend and React frontend.
